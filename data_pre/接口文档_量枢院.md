# 获取期货列表

> 方法名：get_future_list

> 响应参数：

| 字段                  | 类型        | 描述          |
|---------------------|-----------|-------------|
| symbol              | str       | 股票代码        |
| contract_multiplier | str       | 股票名称        |
| de_listed_date      | double    | 退市日期        |
| end_delivery_date   | double    | 结束交割日       |
| exchange            | double    | 交易所         |
| industry_name       | double    | 行业分类名称      |
| listed_date         | double    | 上市日期        |
| margin_rate         | double    | 期货合约的最低保证金率 |
| market_tplus        | double    | 交易制度        |
| maturity_date       | double    | 期货到期日       |
| name                | double    | 期货名称        |
| product             | double    | 合约种类        |
| round_lot           | double    | 期货全部为 1.0   |
| start_delivery_date | double    | 开始交割日       |
| trading_code        | double    | 交易代码        |
| trading_hours       | double    | 可交易时间       |
| type                | double    | 合约类型        |
| underlying_symbol   | double    | 标的名称        |


> 使用示例:

```python  
import panda_data 
result = panda_data.get_future_list()
print(result)
```


# 通用日线获取

> 方法名：get_market_data

> 入参：

| 字段              | 类型                              | 描述                                                                 | 是否必填 |        
|-----------------|---------------------------------|--------------------------------------------------------------------|------|
| start_date      | str                             | 开始日期,eg:"20250702"                                                 | 必填   |
| end_date        | str                             | 结束日期,eg:"20250702"                                                 | 必填   |
| symbol          | Optional[Union[str, List[str]]] | 股票/期货代码                                                            | 非必填  |
| fields          | Optional[Union[str, List[str]]] | 返回字段                                                               | 非必填  |
| symbol_type     | Optional[str]                   | 产品类型，包含："stock" ,"future","index"                                  | 非必填  |
| index_component | Optional[str]                   | 股票池,包含:沪深300:"100",中证500:"010",中证1000:"001" ,仅symbol_type为stock时生效 | 非必填  |

> 响应参数：

股票：

| 字段              | 类型      | 描述     |
|-----------------|---------|--------|
| date            | str     | 日期     |
| symbol          | str     | 股票代码   |
| name            | str     | 股票名称   |
| open            | double   | 当日开盘价  |
| close           | double   | 当日收盘价  |
| high            | double   | 当日最高价  |
| low             | double   | 当日最低价  |
| volume          | double   | 当日成交量  |
| pre_close       | double   | 昨收价    |
| limit_up        | double   | 当日涨停价  |
| limit_down      | double   | 当日跌停价  |
| index_component | str     | 股票池    |
| trade_status    | int     | 当日是否停牌 |

期货：

| 字段                | 类型    | 描述    |
|-------------------|-------|-------|
| date              | str   | 日期    |
| symbol            | str   | 期货代码  |
| dominant_id       | str   | 主力代码  |
| exchange          | str   | 交易所   |
| trading_code      | str   | 交易代码  |
| underlying_symbol | str   | 期货品种  |
| open              | double | 当日开盘价 |
| close             | double | 当日收盘价 |
| high              | double | 当日最高价 |
| low               | double | 当日最低价 |
| volume            | double | 当日成交量 |

指数

| 字段        | 类型     | 描述       |
|-----------|--------|----------|
| symbol    | str    | 指数代码     |
| date      | str    | 日期       |
| open      | double  | 开盘价      |
| close     | double  | 收盘价      |
| high      | double  | 最高价      |
| low       | double  | 最低价      |
| volume    | double  | 成交量      |
| pre_close | double  | 昨日结算价    |
| turnover  | double  | 换手率      |


> 使用示例:

```python  
import panda_data 
market_data_list = panda_data.get_market_data(symbol="A2507", start_date="20250101", end_date="20260110",fields=["open", "high"], symbol_type="future")
market_data_list2 = panda_data.get_market_data(symbol=["A2501", "A2503"],start_date="20250101", end_date="20260101",fields=["open", "high"], symbol_type="future")
print(market_data_list)
```


## 通用分钟线获取

> 方法名：get_market_min_data

> 入参：

| 字段          | 类型                                | 描述                                               | 是否必填 |
|-------------|-----------------------------------|--------------------------------------------------|------|
| symbol      | Optional[str]                     | 股票代码                                             | 非必填  |
| start_date  | str                               | 开始日期,eg:"20250702"                               | 必填   |
| end_date    | str                               | 结束日期,eg:"20250702"                               | 必填   |
| symbol_type | Optional[str]                     | 产品类型，包含："stock" ,"future"                        | 非必填  |
| time_zone   | Optional[tuple]                   | 时间段过滤,格式为("HH:MM", "HH:MM")，例如("10:00", "23:00") | 非必填  |
| fields      | Optional[Union[str, List[str]]]   | 返回字段                                             | 非必填  |

> 使用示例:

```python  
import panda_data  
data = panda_data.get_market_min_data(symbol='A2407',start_date='20240607',end_date='20240607',symbol_type="future")
data2 = panda_data.get_market_min_data(symbol='A2407',start_date='20240607',end_date='20240607',symbol_type="future",time_zone=("9:00","9:30"))

print(data)
print(data2)
```

> 响应参数：

股票：

| 字段         | 类型      | 描述    |
|------------|---------|-------|
| date       | str     | 时间    |
| symbol     | str     | 股票代码  |
| open       | double   | 当日开盘价 |
| close      | double   | 当日收盘价 |
| high       | double   | 当日最高价 |
| low        | double   | 当日最低价 |
| volume     | double   | 当日成交量 |
| num_trades | double   | 成交笔数  |
| amount     | double   | 成交额   |


期货：

| 字段                | 类型      | 描述    |
|-------------------|---------|-------|
| date              | str     | 时间    |
| symbol            | str     | 期货代码  |

| close             | double  | 当日收盘价 |
| high              | double  | 当日最高价 |
| low               | double  | 当日最低价 |
| volume            | double  | 当日成交量 |
| dominant_id       | str     | 主力代码  |
| exchange          | str     | 交易所   |
| trading_code      | str     | 交易代码  |
| trading_date      | str     | 交易日期  |
| underlying_symbol | str     | 期货品种  |

指数

| 字段             | 类型        | 描述     |
|----------------|-----------|--------|
